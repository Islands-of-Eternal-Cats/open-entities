# WASM Bindings PoC (Node) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `wasm-bindings/` workspace crate that exposes `Simulation` (`hello`, `world_json`) to Node via `wasm-pack`, plus a runnable demo and `make wasm-demo`.

**Architecture:** Thin `open_entities_wasm` crate (`cdylib` + `rlib`) owns `open_entities::Api` and forwards to `hello()` and `Api::world_json()`. Serialization stays in `open-entities-lib`; bindings map `ExportError` to `JsValue`. Node demo loads generated glue from `wasm-bindings/pkg/`.

**Tech Stack:** Rust 2024 workspace, `wasm-bindgen 0.2`, `wasm-pack` (Node target), Node.js (ESM via `.mjs`).

**Spec:** `docs/superpowers/specs/2026-05-17-wasm-bindings-poc-design.md`

**Prerequisites (run once per machine):**

```bash
rustup target add wasm32-unknown-unknown
cargo install wasm-pack   # if `wasm-pack` is not on PATH
```

### Task 3: `Simulation` WASM bindings

**Files:**
- Create: `wasm-bindings/src/lib.rs`

- [ ] **Step 1: Create `wasm-bindings/src/lib.rs`**

```rust
use open_entities::{hello, Api, ExportError};
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub struct Simulation {
    api: Api,
}

#[wasm_bindgen]
impl Simulation {
    #[wasm_bindgen(constructor)]
    pub fn new() -> Self {
        Self { api: Api::new() }
    }

    /// Returns the canonical greeting from `open_entities::hello()`.
    pub fn hello(&self) -> String {
        hello().to_owned()
    }

    /// Serializes the ECS world to JSON (schema version 3).
    pub fn world_json(&mut self) -> Result<String, JsValue> {
        self.api
            .world_json()
            .map_err(|e: ExportError| JsValue::from_str(&e.to_string()))
    }
}
```

- [ ] **Step 2: Verify WASM target compiles**

Run (from repo root):

```bash

```

Expected: `Finished` with no errors.

### Task 5: Node demo script

**Files:**
- Create: `wasm-bindings/demo/run.mjs`

- [ ] **Step 1: Create `wasm-bindings/demo/run.mjs`**

```javascript
import init, { Simulation } from "../pkg/open_entities_wasm.js";

await init();
const sim = new Simulation();
console.log(sim.hello());
const json = sim.world_json();
console.log(json);
const parsed = JSON.parse(json);
if (parsed.version !== 3 || !Array.isArray(parsed.entities) || parsed.entities.length !== 0) {
  throw new Error(`unexpected empty world JSON: ${json}`);
}
```
