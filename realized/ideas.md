в файле описания сущностей хранятся сущности с параметрами позиция, скорость

* использовать bevy ecs
* требуется по имени описания типа сущности в файле создавать сущности в ecs
* файлы описания сущностей хранятся в виде yaml
* если получится - создать динамические компоненты, иначе - определить компоненты заранее
* состав компонентов у каждой сущности может быть разным
